# wordNumber

#### 用汉字显示数字


**例如**

##### 李欣远, 520

> wordNumber.print("李欣远", "520");

```
　李李李李　欣欣欣欣　远远远远
　李　　　　　　　欣　远　　远
　李李李李　欣欣欣欣　远　　远
　　　　李　欣　　　　远　　远
　李李李李　欣欣欣欣　远远远远
```
##### 李欣远, 521

> wordNumber.print("李欣远", "520");

　李李李李　欣欣欣欣　　远远

　李李李李　欣欣欣欣　　远远

#### 圆形的零,只支持前四个字符

> wordNumber.print("我爱欣远");

```
　我我我我　爱爱爱爱　  欣远  　
　我　　　　　　　爱　 欣  远 　
　我我我我　爱爱爱爱　欣    远　
　　　　我　爱　　　　 欣  远 　
　我我我我　爱爱爱爱　  欣远  　

```
